Explaining the Semantic Elements Used in the "Coffee Connoisseur" Project

Semantic HTML Layout

The entire website structure relies on modern HTML5 semantic elements to provide meaning to content.

Key structural tags used in all three pages (index.html, videos.html, request.html):

  • <header>: Contains the site name and main title.
  • <nav>: Holds the primary links (Home, Videos, Request) for site navigation.
  • <main>: Encloses the dominant content specific to each page.
  • <section>: Groups related content on the Home Page (e.g., About, Fundamentals, Latest Posts).
  • <article>: Used for self-contained pieces of content, like the individual video entries or blog snippets.
  • <footer>: Contains site metadata, like copyright and contact info.
